Добро пожаловать в API v0.2.5

Текущие ссылки:

/groups - Полный список групп


/teachers - Полный список преподавателей


/shedule/student/%Группа%/today - расписание %Группы% на сегодня УСТАРЕЛО!


/shedule/student/%Группа%/week - расписание %Группы% на неделю УСТАРЕЛО!


/shedule/student/%Группа%/all - все пары %Группы%


/shedule/teacher/%Преподаватель%/today - расписание %Преподавателя% на сегодня


/shedule/teacher/%Преподаватель%/week - расписание %Преподавателя% на неделю


/shedule/teacher/%Преподаватель%/all - все пары %Преподавателя%


Cсылки для бота:

/bot/who?mess=%Мессенджер%&id=%ID% - Кто этот %ID% в этом %Мессенджере%
